Tahap 23 — Favorites / Pin File

Periksa apakah sistem Telegram Drive sudah memiliki fitur Favorites (File Favorit).

Jika fitur ini sudah ada dan berfungsi dengan baik, jangan ubah apa pun.

Jika belum ada, implementasikan fitur berikut dan integrasikan dengan sistem yang sudah ada.

Fitur yang harus ada:

- Tambah/Hapus file ke Favorites.
- Pin file penting agar muncul paling atas.
- Halaman khusus Favorites.
- Filter "Hanya Favorit".
- Pencarian di dalam Favorites.
- Bulk Add Favorites.
- Bulk Remove Favorites.
- Sinkron dengan My Files.
- Tetap kompatibel dengan Search.
- Tetap kompatibel dengan Download.
- Tetap kompatibel dengan Preview.
- Tetap kompatibel dengan Recycle Bin.
- Tetap kompatibel dengan Share.
- Mobile friendly.
- Dark Mode.
- Animasi halus.
- Tidak membuat duplicate code.
- Gunakan komponen yang sudah ada.
- Jangan membuat file .md.
- Jangan mengubah backend jika tidak diperlukan.
- Jangan merusak fitur sebelumnya.

Sebelum selesai lakukan self-test dan pastikan tidak ada error, import rusak, route rusak, maupun JavaScript error.

Jika selesai, tampilkan ringkasan Stage 23 beserta daftar fitur yang berhasil diimplementasikan.
