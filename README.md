Tahap 25 — Duplicate File Detector

Jangan membaca seluruh project atau seluruh dokumentasi agar proses lebih cepat.

Periksa hanya file yang berhubungan dengan Upload, My Files, Search, Favorites, Recycle Bin, Activity, Download, Dashboard, dan sistem penyimpanan file.

Jika fitur Duplicate File Detector sudah ada dan berjalan dengan baik, jangan membuat ulang.

Jika belum ada, implementasikan fitur berikut dan integrasikan dengan sistem yang sudah ada.

Fitur yang harus tersedia:

• Mendeteksi file duplikat berdasarkan hash/file id.
• Menampilkan daftar file yang dianggap duplikat.
• Menampilkan jumlah duplikat.
• Preview sebelum menghapus.
• Pilih file mana yang ingin dipertahankan.
• Bulk delete duplicate.
• Bulk move duplicate ke Recycle Bin.
• Filter hanya file duplikat.
• Search pada daftar duplicate.
• Integrasi dengan Search.
• Integrasi dengan My Files.
• Integrasi dengan Favorites.
• Integrasi dengan Recent Files.
• Integrasi dengan Recycle Bin.
• Integrasi dengan Dashboard.
• Mobile friendly.
• Dark mode.
• Animasi halus.
• Tidak membuat duplicate code.
• Gunakan komponen yang sudah ada.
• Jangan membuat file .md.
• Jangan mengubah backend jika tidak diperlukan.
• Jangan merusak fitur sebelumnya.

Sebelum selesai lakukan self-test dan pastikan tidak ada error, import rusak, route rusak, JavaScript error maupun konflik dengan fitur sebelumnya.

Tampilkan ringkasan Stage 25 setelah selesai.
