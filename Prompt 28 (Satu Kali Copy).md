Tahap 28 — Link Analytics & Statistik Download

Jangan membaca seluruh project atau dokumentasi agar proses lebih cepat.

Periksa hanya file yang berhubungan dengan Download, Share, Public Link, Dashboard, Activity, Search, dan File Details.

Jika fitur Link Analytics sudah ada dan berjalan dengan baik, jangan membuat ulang.

Jika belum ada, implementasikan fitur berikut dan integrasikan dengan sistem yang sudah ada.

Fitur yang harus tersedia:

• Statistik jumlah download setiap file.
• Statistik jumlah view/open link.
• Last download.
• Last opened.
• Download trend.
• File paling populer.
• Top 10 file paling banyak di-download.
• Statistik per hari.
• Statistik per minggu.
• Statistik per bulan.
• Halaman Analytics.
• Search.
• Filter.
• Export statistik (CSV/JSON jika memungkinkan).
• Integrasi dengan Dashboard.
• Integrasi dengan Public Link.
• Integrasi dengan My Files.
• Integrasi dengan Activity.
• Mobile friendly.
• Dark mode.
• Animasi halus.
• Tidak membuat duplicate code.
• Gunakan komponen yang sudah ada.
• Jangan membuat file .md.
• Jangan mengubah backend jika tidak benar-benar diperlukan.
• Jangan merusak fitur sebelumnya.

Sebelum selesai lakukan self-test dan pastikan tidak ada error, import rusak, route rusak, JavaScript error maupun konflik dengan fitur sebelumnya.

Tampilkan ringkasan Stage 28 setelah selesai.
