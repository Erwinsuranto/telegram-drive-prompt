Tahap 26 — Storage Cleanup & Smart Storage

Jangan membaca seluruh project atau dokumentasi agar proses lebih cepat.

Periksa hanya file yang berhubungan dengan Dashboard, My Files, Storage, Search, Recycle Bin, Duplicate File Detector, Upload, dan Activity.

Jika fitur Storage Cleanup sudah ada dan berjalan dengan baik, jangan membuat ulang.

Jika belum ada, implementasikan fitur berikut dan integrasikan dengan sistem yang sudah ada.

Fitur yang harus tersedia:

• Halaman Storage Cleanup.
• Menampilkan total storage yang digunakan.
• Menampilkan jumlah file.
• Menampilkan ukuran rata-rata file.
• Menampilkan file terbesar.
• Menampilkan folder terbesar.
• Menampilkan file yang sudah lama tidak diakses.
• Menampilkan file yang tidak pernah di-download.
• Menampilkan rekomendasi file yang dapat dipindahkan ke Recycle Bin.
• Tombol Quick Cleanup.
• Tombol Select All Recommendation.
• Bulk Move to Recycle Bin.
• Estimasi ruang yang akan dibebaskan.
• Filter berdasarkan ukuran.
• Filter berdasarkan umur file.
• Search.
• Integrasi dengan Dashboard.
• Integrasi dengan My Files.
• Integrasi dengan Recycle Bin.
• Integrasi dengan Duplicate File Detector.
• Mobile friendly.
• Dark mode.
• Animasi halus.
• Tidak membuat duplicate code.
• Gunakan komponen yang sudah ada.
• Jangan membuat file .md.
• Jangan mengubah backend jika tidak benar-benar diperlukan.
• Jangan merusak fitur sebelumnya.

Sebelum selesai lakukan self-test dan pastikan tidak ada error, import rusak, route rusak, JavaScript error maupun konflik dengan fitur sebelumnya.

Tampilkan ringkasan Stage 26 setelah selesai.
